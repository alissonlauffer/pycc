# PyCC - Python Compiled Compiler

PyCC is a Python interpreter and compiler that generates LLVM IR and compiles to native code.

## Features
- Python-like syntax for ease of use
- Dynamic typing with compilation optimizations
- LLVM-based backend for high-performance code generation

## Project Structure
- `src/` - Compiler source code
- `docs/` - Documentation
- `tests/` - Test suite
- `examples/` - Example programs
